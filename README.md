# Bill Splitter

[[`Bill Splitter Site`](https://billing.jonathanlawhh.com/) [`My Website`](https://jonathanlawhh.com/)]

Just to split my bills among my friends. Partially built using Antigravity.

## Setup

Make sure to install dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```
